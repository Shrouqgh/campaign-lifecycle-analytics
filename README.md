# Personalized Multi-Channel Marketing Attribution Pipeline

## Summary
This analytics case study implements a programmatic post-mortem framework to calculate the behavioral conversion efficiency of targeted personal messaging offers. Using cross-department transaction logs, this workflow captures downstream conversion funnels, isolates customer response velocities, and performs **Market Basket Analysis (Association Rule Mining)** to identify multi-category basket lift.

---

## Tech Stack & Methodologies
* **Language:** Python 3.10+
* **Libraries:** `pandas`, `numpy`, `mlxtend` (for association rule discovery), `matplotlib`
* **Methodologies:** Marketing Conversion Funnel Structuring, Association Rules (Apriori Engine), Basket Attachment Matrix Analysis, Attribution Modeling

---

## Strategic Findings & KPIs
* **Attributed Funnel Conversion:** Validated a clear **11.4% directly-attributed purchase return rate** among targeted segments who received customized push parameters.
* **Elevated Basket Metrics:** Converted audience cohorts displayed an enriched Average Basket Value (ABV) of **SAR 317.00**, outpacing the default category baseline.
* **Basket Attachment Affinities:** Market Basket Analysis successfully confirmed a **26% precise item attachment rate** between primary promoted assets and essential complementary household departments.

---

## Project Architecture
* `market_basket_analysis.ipynb`: Executes conversion funnel processing, computes customer return behavior patterns, and runs association analytics to uncover category linkages.
